Para poder escribir un texto o un párrafo dentro de HTML se utiliza la etiqueta p, y para dar un salto de línea se utiliza la etiqueta br
Al colocar la etiqueta br el texto que este después va a aparecer en una nueva línea dentro de la página


Para dar más espacio entre líneas se pueden colocar varios br junto, pero esto es mala práctica

Para dar una separación entre párrafos o secciones de la página con una línea se puede usar hr
Para comentar una sección de la página con el teclado se utiliza la combinación de Alt + Shift + A
Los comentarios sirven para entender la estructura que se está trabajando en la página y el usuario no los ve

Ahora se van a probar tres etiquetas para editar texto en HTML:
La etiqueta strong, que coloca en negrilla el texto que esté dentro de ella.
La etiqueta em, la cual coloca todo el texto que esté dentro de ella en cursiva.
Y la etiqueta span, la cual ayuda a seleccionar cierta parte en un texto para editar solo esa parte (se usó un poco de CSS para mostrar mejor esta etiqueta).

Otra etiqueta a usar es la etiqueta a href="" que nos ayuda a insertar hipervínculos.

Y en las comillas de href se coloca el enlace de destino.

En esta parte se hará uso de la etiqueta img src="" alt="" la cual permite colocar imágenes en una página web donde:
En src va el enlace o la ubicación de la imagen.
Y en alt irá un texto que aparecerá en dado caso la imagen no cargue.
Cabe decir que la etiqueta img soporta varios formatos de archivos tipo imagen como: JPEG, PNG y GIF.
