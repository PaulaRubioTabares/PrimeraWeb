Para poder escribir un texto o un párrafo dento de html se utiliza la etiqueta p, y para dar un salto de linea se utiliza la etiqueta br
Al colocar la etiqueta br el texto que este despues va a aparecer en una nueva linea dentro de la página


Para dar más espacio entre lineas se pueden colocar varios br junto, pero esto es mala práctica

Para dar una separación entre párafos o secciones de la página con una linea se puede usar hr
Para comentar una sección de la pagina con el teclado se utiliza la combinación de Alt + Shift + A
Los comentarios sirven para entender la estructura que se esta trabajándo en la página y el usuario no los ve
