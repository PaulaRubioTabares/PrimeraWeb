Para poder escribir un texto o un párrafo dentro de HTML se utiliza la etiqueta p, y para dar un salto de línea se utiliza la etiqueta br
Al colocar la etiqueta br el texto que este después va a aparecer en una nueva línea dentro de la página


Para dar más espacio entre líneas se pueden colocar varios br junto, pero esto es mala práctica

Para dar una separación entre párrafos o secciones de la página con una línea se puede usar hr
Para comentar una sección de la página con el teclado se utiliza la combinación de Alt + Shift + A
Los comentarios sirven para entender la estructura que se está trabajando en la página y el usuario no los ve

Ahora se van a probar tres etiquetas para editar texto en HTML:
La etiqueta strong, que coloca en negrilla el texto que esté dentro de ella.
La etiqueta em, la cual coloca todo el texto que esté dentro de ella en cursiva.
Y la etiqueta span, la cual ayuda a seleccionar cierta parte en un texto para editar solo esa parte (se usó un poco de CSS para mostrar mejor esta etiqueta).

Listas no ordenadas

  • Esté sería el primer elemento de la lista no ordenada
  • Este sería el segundo elemento
  • Este el tercero
  • Y así sucesivamente

Una característica de estas listas es que con un poco de CSS se puede cambiar el tipo de viñeta a usar, en este caso dentro de htl y en la etiqueta ul usando: style="list-style-type: circle;"

  • Este el tercero
  • Y así sucesivamente

Tablas

En este espacio se introduce el tema de tablas en html, indicando que etiquetas se deben utilizar para crearlas y añadir filas y columnas.

  • Para empezar a crear una tabla se utiliza la etiqueta table.
  • Para definir la primera fila se utiliza la etiqueta tr.
  • Dentro de esta primera etiqueta tr se usa un th por cada columna de la tabla. En este caso th es para indicar también que esa fila va a ser la cabecera de la tabla, el contenido va a aparecer centrado y en negrilla
  • Para las columnas de las siguientes filas, que serán el contenido de la tabla, se usa td.

Para que la tabla quede organizada se deben tener la misma cantidad de etiquetas th y td en cada tr, para que no sobren o falten columnas como se ve a continuación
